# OpenDataKBB

## Overview

`OpenDataKBB` is a project designed to provide open access to data related to the KBB (Kota Bandung Barat) region. The repository includes both frontend and backend components to facilitate data visualization and management.

---

## Repository Structure

- **`BackEnd/`**: Contains the backend code and server-side logic.
- **`my-app/`**: The frontend application built to interact with the backend and display data.

---

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/Khorzyy/OpenDataKBB.git
cd OpenDataKBB
```
## 2. Backend Setup
Navigate to the BackEnd/ directory and install dependencies:

```bash
Copy code
cd BackEnd
npm install
```
Start the backend server:
```bash
Copy code
npm start
```
## 3. Frontend Setup
Navigate to the my-app/ directory and install dependencies:

```bash
Copy code
cd ../my-app
npm install
Start the frontend application:
```

```bash
Copy code
npm start
The frontend should now be accessible at http://localhost:3000, and it will interact with the backend server running on http://localhost:5000.
```

## Technologies Used
- **`Backend`**: Node.js with Express
- **`Frontend`**: React.js
- **`Styling`**: CSS
